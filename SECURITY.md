# Security Policy

## Supported Versions

Currently, the following versions of code-context are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

*Note: Since the project is in early development, only the latest minor release is officially supported for security updates.*

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **do not** report it by creating a public GitHub issue. 

Instead, please send an email to the repository maintainers or use GitHub's private vulnerability reporting feature if enabled on the repository.

1. Go to the **Security** tab of the repository.
2. Click on **Advisories**.
3. Click on the **Report a vulnerability** button.

We take security seriously and will try to respond to reports within 48 hours. When reporting, please provide:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

**Thank you for helping keep this project secure!**
